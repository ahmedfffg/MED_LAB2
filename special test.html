اختر الباقة المناسبة لك واستمتع بتحاليل إضافية مجانية أو خصومات مميزة – بدون أسعار، فقط رعاية صحية متكاملة

باقات متكاملة

باقة الجهاز الهضمي

عرض خاص
  • تحليل جرثومة المعدة (H.pylori)
  • زراعة براز + حساسية
  • اختبار الدم الخفي في البراز
هدية مجانية: تحليل بول كامل + زراعة بول (قيمة الفحص مجاناً مع الباقة)
استفسر عن العرض

الباقة الأيضية المتكاملة

توفير كبير
  • السكر الصائم (FBS)
  • الدهون الثلاثية والكوليسترول الكلي
  • وظائف الكلى (يوريا، كرياتينين)
  • وظائف الكبد (ALT, AST, ALP)
سعر مميز للباقة (أقل من إجراء كل تحليل على حدة) + هدية تحليل فيتامين د
احجز الباقة الآن

باقة الطاقة والهرمونات

عرض لفترة محدودة
  • TSH, T3, T4 (وظائف الغدة الدرقية)
  • فيتامين B12, فيتامين D
  • هرمون الحديد (Ferritin)
هدية: صورة دم كاملة (CBC) مجاناً مع الباقة

تحليل بول + براز هدية

عند إجراء تحليل جرثومة المعدة بأي نوع (تنفس أو براز)

مجاناً بالكامل

صورة دم كاملة مجاناً

مع أي باقة كبد أو كلى شاملة (وظائف كبد + كلى + دهون)

هدية تلقائية

تحليل فيتامين D مجاني

عند إجراء تحليل السكر التراكمي + الدهون الثلاثية معاً

عرض محدود

باقات مركبة (سعر خاص)

باقة السكر والقلب

✔ السكر الصائم + السكر التراكمي
✔ الدهون الثلاثية + الكوليسترول الضار والنافع
✔ وظائف الكبد (SGOT, SGPT)

🎁 هدية: تحليل بول كامل + قياس ضغط الدم مجاناً

باقة الكلى والكبد المثالية

✔ وظائف الكلى (كرياتينين، يوريا، حمض اليوريك)
✔ وظائف الكبد الكاملة (ALT, AST, GGT, البيليروبين)
✔ تحليل الألبومين والبروتين الكلي

🎁 هدية: تحليل فيتامين B12 مجاناً

باقة الفحص الشامل

✔ سكر صائم + دهون ثلاثية + كوليسترول
✔ وظائف كلى + كبد + صورة دم كاملة
✔ جرثومة المعدة (اختياري بسعر رمزي)

🎁 هدية إضافية: تحليل براز + بول هدية مزدوجة

عرض خاص جداً: إجراء جرثومة المعدة (بأي تقنية)

احصل على تحليل بول كامل + تحليل براز (زراعة وفحص طفيليات) هدية مجانية – بدون أي رسوم إضافية! العرض ساري للعائلات أيضاً.

أسئلة عن العروض والباقات

هل يمكن الجمع بين أكثر من عرض أو هدية؟
نعم في بعض الباقات الشاملة، يمكن الاستفادة من أكثر من هدية حسب سياسة المعمل. يفضل التواصل مع خدمة العملاء للتأكد من أفضل عرض يناسب احتياجك.
كيف أحصل على تحليل البول والبراز مجاناً مع جرثومة المعدة؟
ما عليك سوى طلب تحليل جرثومة المعدة (تنفس أو براز) وسيتم إضافة تحليل البول الكامل والبراز الروتيني كهدية تلقائياً بدون أي رسوم إضافية خلال فترة العرض.
باقة السكر الصائم + الدهون الثلاثية + وظائف الكلى والكبد بسعر أقل: كيف تعمل؟
هذه الباقة مصممة لدعم متابعة مرضى السكر والضغط. تجمع التحاليل الأساسية بسعر مخفض مقارنة بإجرائها بشكل منفصل، ويتم إضافة هدية تحليل فيتامين د أو صورة دم حسب الباقة الفرعية.
هل العروض تشمل جميع فروع ميدلاب؟
نعم، العروض سارية في جميع فروع ميدلاب حتى إشعار آخر. بعض الباقات تتطلب حجزاً مسبقاً عبر الموقع أو الاتصال بخدمة العملاء.
استشر طبيبك قبل اختيار الباقة المناسبة لحالتك الصحية. فريق ميدلاب يسعد بتقديم الاستشارة المبدئية مجاناً.
